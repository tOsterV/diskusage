# DiskUsage CLI

Консольная утилита для анализа использования дискового пространства. Показывает соотношение занятого и свободного места на диске.

## Функциональные требования

### 1. Запуск и аргументы
- Запуск из командной строки
- Принимает путь для анализа в качестве аргумента: `diskusage C:\Projects`
- Если аргумент не указан - анализируется текущая директория (Весь том диска)

### 2. Сбор данных
- Рекурсивный обход всех файлов и папок по указанному пути
- Подсчет общего размера папки
- Подсчет количества файлов
- Определение диска, на котором находится целевая папка

### 3. Системная информация
- Получение общего объема диска
- Получение занятого пространства на диске
- Получение свободного пространства на диске

### 4. Вывод информации
Программа выводит в консоль:
```
DISK USAGE REPORT:

--Info about disk [C:] --

Общий объем диска:??GB

Занято на диске: ??GB

Свободно на диске: ??GB
```
## Нефункциональные требования

### 1. Производительность
- Быстрое сканирование директорий
- Корректная обработка больших объемов данных

### 2. Читаемость
- Вывод размеров в человеко-читаемом формате (Б, КБ, МБ, ГБ, ТБ)
- Выравнивание и форматирование для удобного восприятия
- Разделители тысяч для больших чисел

### 3. Надежность
- Обработка ошибок доступа к папкам/файлам без падения программы
- Информирование пользователя о пропущенных элементах
- Корректная работа с путями, содержащими пробелы и спецсимволы

### 4. Кроссплатформенность (в зависимости от задания)
- Поддержка целевой ОС (Windows/Linux/macOS)
- Корректная обработка путей в формате целевой ОС
